Week 3 Dev Blog

Forms Advanced CSS Layouts

  • I learnt how to use the <form></form>tag to make a simple form.
  • I learnt how to use the <input>tag such as
    1. <input type="text">
    2. <input type="date">
    3. <input type="color">
    4. <input type="file">
    5. <input type="checkbox">
    6. <input type="radio">
    7. <input type="submit">
    The "type" attribute determines the type of input.
  • I learnt how to use the <label></label>tag to make the site accessible.
  • I learnt how to write simple validations to make the required input section not empty using the "required" attribute.
  • I learnt Specificity that determines how browsers decide which CSS rule takes precedence.
  • I learnt how to use Advanced positioning in CSS like Relative, Absolute and Fixed Position.
  • I learnt to apply flexbox to get flexible layouts.
  • I learnt about wireframes.
